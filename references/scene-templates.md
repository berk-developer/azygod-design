## 1. WeChat Kapak / Makale Başlık Resmi

**Özellikler**:
- Kapak resmi: 2.35:1 (900×383px veya 1200×510px)
- Makale iç resim: 16:9 (1200×675px) veya 4:3 (1200×900px)

**Kilit tasarım öğeleri**:
- Görsel etki öncelikli (kullanıcı bilgi akışında hızlıca kaydırır)
- Çok az yazı veya yazısız (WeChat başlığı üzerine kapanır)
- Renk doygunluğu orta (WeChat okuma ortamı beyaz tonludur)
- Aşırı detaydan kaçın (küçük resimde de tanınabilir)

**Önerilen stiller**: 01 Pentagram / 11 Build / 12 Sagmeister / 18 Kenya Hara / 07 Field.io

**Sahne ipucu şablonu**:
```
[Stil DNA'sı buraya]
- Article cover image for WeChat subscription
- Landscape format, 2.35:1 aspect ratio
- Bold visual impact, minimal or no text
- Moderate color saturation for white reading environment
- Must remain recognizable as thumbnail
- Clean composition with clear focal point
```

## 2. Makale İç Resim / Kavram İllüstrasyonu

**Özellikler**:
- 16:9 (1200×675px) en genel
- 1:1 (800×800px) vurgu için uygun
- 4:3 (1200×900px) bilgi yoğun için uygun

**Kilit tasarım öğeleri**:
- Makale tezine hizmet, dekorasyon değil
- Bağlamla görsel ritim oluştur
- Tek bir çekirdek kavramı özlü ifade et
- AI üretimi öncelikli, HTML ekran görüntüsü yalnızca hassas veri tablolarında kullan

**Önerilen stiller**: Makale tonuna göre seç, sıklıkla 01/04/10/17/18

**Sahne ipucu şablonu**:
```
[Stil DNA'sı buraya]
- Article illustration, concept visualization
- [16:9 / 1:1 / 4:3] aspect ratio
- Single clear concept: [çekirdek kavramı açıkla]
- Serve the argument, not decoration
- [Light/Dark] background to match article tone
```

## 3. İnfografik / Veri Görselleştirme

**Özellikler**:
- Dikey uzun resim: 1080×1920px (mobil okuma)
- Yatay: 1920×1080px (makale iç gömme)
- Kare: 1080×1080px (sosyal medya)

**Kilit tasarım öğeleri**:
- Bilgi hiyerarşisi net (başlık → çekirdek veri → detay)
- Veri doğru, uydurma değil
- Görsel yönlendirme çizgisi (kullanıcı okuma yolu)
- İkon/grafik anlamayı desteklemek için uygun kullan

**Önerilen stiller**: 04 Fathom / 10 Müller-Brockmann / 02 Stamen / 17 Takram

**Sahne ipucu şablonu**:
```
[Stil DNA'sı buraya]
- Infographic / data visualization
- [Vertical 1080x1920 / Horizontal 1920x1080 / Square 1080x1080]
- Clear information hierarchy: title → key data → details
- Visual flow guiding reader's eye path
- Icons and charts for comprehension
- Data-accurate, no decorative distortion
```

## 4. PPT / Keynote Sunum

**Özellikler**:
- Standart: 16:9 (1920×1080px)
- Geniş ekran: 16:10 (1920×1200px)

**Kilit tasarım öğeleri**:
- Her sayfa bir çekirdek bilgi (yığma değil)
- Yazı boyutu hiyerarşisi net (başlık 40pt+ / gövde 24pt+ / not 16pt+)
- Bol boşluk, projeksiyonda daha net
- Görsel-metin oranı en az 60:40
- Tutarlı görsel sistem (renk, yazı tipi, aralık)

**Önerilen stiller**: 01 Pentagram / 10 Müller-Brockmann / 11 Build / 18 Kenya Hara / 04 Fathom

**Sahne ipucu şablonu**:
```
[Stil DNA'sı buraya]
- Presentation slide design, 16:9
- One core message per slide
- Clear type hierarchy (title 40pt+, body 24pt+)
- Generous whitespace for projection clarity
- Consistent visual system throughout
- [Light/Dark] theme
```

## 5. PDF Beyaz Kitap / Teknik Rapor

**Özellikler**:
- A4 dikey (210×297mm / 595×842pt)
- Letter dikey (216×279mm / 612×792pt)

**Kilit tasarım öğeleri**:
- Uzun metin okuma optimizasyonu (satır genişliği 66 karakter, satır yüksekliği 1.5-1.8)
- Net bölüm navigasyon sistemi
- Üstbilgi/altbilgi/sayfa numarası birleşik tasarım
- Grafik ve gövde metninin zarif bir arada varlığı
- Alıntı/not sistemi
- Kapak sayfası tasarımı incelikli

**Önerilen stiller**: 10 Müller-Brockmann / 04 Fathom / 03 Information Architects / 17 Takram / 19 Irma Boom

**Sahne ipucu şablonu**:
```
[Stil DNA'sı buraya]
- PDF document / white paper design
- A4 portrait format (210×297mm)
- Long-form reading optimized (66 char line width, 1.5 line height)
- Clear chapter navigation system
- Elegant header/footer/page number design
- Charts integrated with body text
- Professional cover page
```
